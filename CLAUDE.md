# Ayaan Merchant — Portfolio Website

## Project Overview
Personal portfolio website deployed at **ayaanmerchant.me** via GitHub Pages.
Repository: AyaanMerchant.github.io

## Tech Stack
- Static site (HTML/CSS/JS)
- Three.js for 3D animations and visual effects
- Hosted on GitHub Pages with custom domain (CNAME: ayaanmerchant.me)

## Development Guidelines
- Keep everything performant — this is a portfolio, first impressions matter
- Mobile-responsive design is mandatory
- Prefer vanilla JS where possible to keep bundle size small; use CDN for libraries like Three.js
- All assets should be optimized for web (compressed images, minified where appropriate)
- Maintain a clean, modern, and professional aesthetic with the indigo/violet color palette

## Structure
- `index.html` — Main entry point (currently a Coming Soon page)
- `CNAME` — Custom domain config for GitHub Pages

## Deployment
- Push to `main` branch auto-deploys to GitHub Pages
- Live at: https://ayaanmerchant.me
